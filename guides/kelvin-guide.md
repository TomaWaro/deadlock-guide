## Section 1: AP Priority (Ordre des Sorts)
### Content
**Priorité:** Arctic Beam (3) → Frozen Shelter (4) → Grenades après

**Build Philosophy:** Contrairement aux builds classiques qui maxent la grenade ou le dôme, ce build se concentre sur le rayon pour le DPS et le dôme pour l'utilité complète.

**AP Allocation:**
- 3 points: Arctic Beam (devient votre source principale de dégâts)
- 2 points: Frozen Shelter (contrôle de terrain + utility)
- 1 point: Grenades (heal seulement, pas damage)

**Key Difference:** Les grenades sont utilisées UNIQUEMENT pour soigner, pas pour infliger des dégâts.

### Media
- YouTube: https://www.youtube.com/watch?v=Mtfx2rcvBzQ?t=0

### Tips
- N'utilisez pas vos points pour les grenades en priorité
- Le rayon devient votre source principale de dégâts dès le Tier 2
- Grenades = heal tool, pas damage tool

## Section 2: Early Game Core (Laning Phase)
### Content
**Items Essentiels (Lane Phase):**

1. **Guardian Ward** 
   - Essentiel pour la survie de votre équipe sur la ligne
   - Place early, protège tout le positionnement

2. **Restorative Barrel**
   - Donne -35% fire rate aux ennemis ciblés
   - CUMULE avec les -40% de votre Arctic Beam
   - Résultat: Les ennemis ne peuvent VIRTUELLEMENT PLUS TIRER quand vous les visez
   - Permet de "jump" sur les ennemis et les neutraliser complètement

3. **Extra Charge**
   - Charge supplémentaire de grenade de soin
   - Sustain augmenté en lane

**Playstyle Lane:**
- Guardian Ward = Protection passive équipe
- Restorative Barrel = Offensive utility (debuff ennemi)
- Grenades = Sustain heal uniquement
- Arctic Beam = Pressure damage

### Media
- YouTube: https://www.youtube.com/watch?v=Mtfx2rcvBzQ?t=15

### Tips
- Les grenades sont utilisées uniquement pour le soin, pas pour les dégâts
- Restorative Barrel + Arctic Beam slow = ennemi totalement neutralisé
- Utilisez le dôme pour les plays défensives seulement en lane

## Section 3: Mid-Game Power Spike (Burst Spirit)
### Content
**Combo Principal: Arcane Surge + Tier 2 Arctic Beam**

**Effet:** Une fois Rayon au Tier 2 + Arcane Surge achetés = ~800 damage burst rapide

**Sequence d'Exécution:**
1. Activez Arcane Surge juste AVANT de lancer Arctic Beam (touche 3)
2. Lancez le rayon immédiatement après
3. Dégâts explosent avec Arcane Surge multiplier

**Items Mid-Game:**
- **Arcane Surge** (Premier item Spirit) = Burst multiplier
- **Mystic Vulnerability** = Debuff ennemis, augmente vos dégâts
- **Healing Booster** = Augmente régénération PV dans le dôme
- **Rapid Recharge** = Spam supplémentaire grenades heal

**Timing:**
- Vous n'êtes INVULNÉRABLE qu'avec le dôme
- Exploitez le Tier 2 Arctic Beam pour burst quand dôme actif
- Utilisez Rapid Recharge pour maintenir grenade cooldown bas

- Activez Arcane Surge AVANT de lancer rayon = timing critique

## Section 4: Late Game Mastery (Double Dome Strategy)
### Content
**La Stratégie "Double Dôme" = Win Condition:**

**Étape 1: Warpstone**
- Permet téléportation instantanée
- Utilisé pour placer dôme parfait et sauver allié
- Activation tactique de position

**Étape 2: Refresher**
- L'item ULTIME pour Kelvin
- Séquence: Posez dôme 1 → Attendez fin → Utilisez Refresher → Posez dôme 2
- Double dôme coverage = zone de contrôle immense

**Étape 3: Expansions**
- **Expansion Items:** Agrandissez zone dôme
- **Duration Items:** Rallongez durée dôme
- Résultat: Dôme couvre zone MASSIVE pour durée LONGUE

**Item Priority (Late Game):**
1. Warpstone (teleport positioning)
2. Refresher (double dôme enabler)
3. Superior Cooldown / Transcendent Cooldown (cooldown reduction)
4. Expansion (dôme radius)
5. Duration (dôme longevity)

**Double Dôme Execution:**
- Dôme 1: Cover team fight initial
- Refresher ready: Use immediately after Dome 1 ends
- Dôme 2: Re-cover pour teamfight extension
- Result: 20+ secondes de protection continue (2 dômes back-to-back)

### Media
- YouTube: https://www.youtube.com/watch?v=Mtfx2rcvBzQ?t=80

### Tips
- Double dôme permet de gagner PRESQUE N'IMPORTE QUEL combat autour du Mid Boss
- Superior + Transcendent Cooldown = essential pour speedrun Refresher
- Placement Warpstone = instant teleport pour dôme parfait
- Dôme coverage = meilleure défense Kelvin

## Section 5: Adaptability & Situational Utility
### Content
**Core Build Fixe (Sections 1-4), Puis Adapter:**

Après build noyau de base, achetez items utilitaires SELON LES ENNEMIS:

**Against High Healing Enemies:**
- **Heal Bane:** Réduit efficacité heal ennemi
- Priorité: Si ennemi héros soigneur ou self-heal high

**Against Mobile Heroes:**
- **Slowing Hex:** Slow supplémentaire pour catch
- Priorité: Versus Ivy, ultra-mobiles

**Against Flying Heroes:**
- **Knockdown:** Knockback pour ground flying ennemis
- Priorité: Versus Warden, flying characters

**Against Gun-Based Heroes:**
- **Disarming Hex:** Désarme tirs directs (M1)
- Priorité: Versus gun-heavy comps (Haze, etc)

**Late Game Survival:**
- **Diviner's Kevlar:** Massive cleanse + armor
- Priorité: Late game 5v5 fights, cleanse CC chains

**Wave Management:**
- **Fix the Waves:** Manage minion waves
- Technique: Maintain pressure, don't let enemies scale free

**Decision Making:**
- Ne suivez pas liste rigide
- Regardez 5 ennemis
- Achetez le counter spécifique
- Flex build per situation

### Media
- YouTube: https://www.youtube.com/watch?v=Mtfx2rcvBzQ?t=120

### Tips
- Regardez les héros adverses AVANT d'acheter utility items
- Hex items > Spirit items si matchup difficile
- Diviner's Kevlar = late game insurance
- Wave management = maintenir pression economique

## Section 7: Playstyle & Role Understanding
### Content
**Analogie:** Jouer ce build Kelvin = Être architecte de champ de bataille

Contrairement aux joueurs qui jettent grenades au hasard et posent dômes réactifs:

**Your Approach:**
- Construire zones de sécurité IMPÉNÉTRABLES (dômes)
- Utiliser rayon comme LASER DE PRÉCISION pour damage
- Soigner alliés strategiquement avec grenades

**Decision Framework:**
1. **Lane Phase:** Guardian Ward protection + Restorative Barrel oppression
2. **Mid Game:** Arcane Surge burst combos when teammate in danger
3. **Late Game:** Double dôme control = teamfight orchestration
4. **Throughout:** Grenades = heal utility, not damage source

**Team Win Condition:**
- Équipe protégée dans dôme (double if possible)
- Ennemis slow + debuff (Restorative + Mystic)
- Dégâts massifs via Arctic Beam spam
- Victory = Field control mastery

### Media
- YouTube: https://www.youtube.com/watch?v=Mtfx2rcvBzQ

### Tips
- Kelvin = Support carry hybrid (not pure support)
- Priorité: Dôme placement > Grenade spam
- Arctic Beam = main damage tool (maximize it)
- Be the architect, not the grenadier
